# 전화번호 관리 프로그램

간단한 전화번호 관리 프로그램

- 전화번호를 관리
- 전화번호는 타인의 계정을 추가할 때 같이 추가됨

# 사용한것

- Qt 6.7.3
- QtCreator
- mysql 8.0
- QMySql(설치 : https://www.youtube.com/watch?v=7YIJUL9pz2U)

# 기능
- 로그인, 로그아웃, 회원가입, 비밀번호 찾기
- 테이블에 타유저의 계정 저장 기능(전화번호도 같이 저장됨)
- 테이블 행 삭제
- 테이블에 저장되어 있는 타유저의 프로필 보기 기능
- 프로필 페이지에서 계정 비밀번호 변경 기능
